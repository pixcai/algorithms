### 解法一
内置方法`isNaN`。由于空白符被判断为`false`，所以需额外处理。

### 解法二
正则表达式：

- `\d+`
匹配`123`等。
- `\.?\d+(?:e[+-]?\d+)?`
匹配`.123`、`.12e3`、`.12e+3`、`.12e-3`、`12e3`、`12e+3`、`12e-3`等。
- `\d+\.\d*(?:e[+-]?\d+)?`
匹配`123.`、`1.23`、`1.2e3`、`1.2e+3`、`1.2e-3`、`1.e23`、`1.e+23`、`1.e-23`等。
